Hi there!

Your recipe is ready — now let's get it on your store. Add the Recipe Kit section to your blog template and you're one click away from publishing.

Add Recipe Kit to your blog template!

Get set up in under 2 minutes:

1.  Open Theme Editor
2.  Select Blog posts → Default blog post
3.  Click Add section in the Template area
4.  Under Apps, select Recipe Kit
5.  Click Save
Install on your theme

Quick tip: No code, no developers needed. You can always adjust the layout or add custom CSS later!
