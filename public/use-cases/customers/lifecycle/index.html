One profile per customer

  • Unified on signup
    Every new user becomes a single profile automatically — no ingestion pipelines, no duplicate records, no manual cleanup.
  • Traits & attributes
    Store any data you care about — plan, referrer, onboarding step — and query it anywhere in the product.
  • Shared across systems
    Billing, messaging, analytics, and automations all read and write to the same customer record — no sync jobs.
