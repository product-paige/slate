Real workflows

From first user to ongoing growth.

Capture and track new users

  • Profiles on signup
    New users automatically get a unified profile — no ingestion pipelines, no duplicate records, no manual cleanup.
  • User attributes & metadata
    Store any data you care about — plan, referrer, onboarding step — and query it anywhere in the product.
  • Lifecycle tracking
    Watch users move from new → active → paid → churned, all in one view your whole team can use.

Segment users for messaging

  • Group by behavior
    Build segments from any attribute or event — "signed up last 7 days," "hit paywall twice," "never opened the app."
  • Wire to campaigns
    Point any segment at an onboarding sequence, re-engagement email, or in-app nudge — no glue code required.
  • Live sync, everywhere
    Segments update in real time as users qualify or drop out — your messaging always hits the right audience.

Track user activity in real time

  • Key event logging
    Capture signup, activation, upgrade, churn — and every custom event that matters to your product.
  • Lifecycle progression
    See activation rates, time-to-value, and retention curves without stitching together three separate tools.
  • Behavior-driven automations
    Trigger workflows the moment a user hits (or misses) a milestone — no overnight batch jobs, no lag.
